# ブログ作者への許諾メール 要否の検討

- **日付**: 2026-06-20
- **問い**: ルールを大幅変更した今、参考元ブログ（boardgameblog.net）の作者に許諾メールを送るべきか
- **結論**: **許諾メールは送らない**。法的に不要かつ逆にリスク。お礼の一報のみ任意。

## 背景

- 本アプリは当初、boardgameblog.net「オープンフェイス・チャイニーズポーカー・ターボ」
  （http://boardgameblog.net/2022/06/02/ocpt/ ）のルールをほぼ忠実に実装していた（ADR-0001）。
- その後、レート戦の品質と差別化のためルールを大幅変更：
  - 毎ストリート1枚捨て導入
  - 全モード ヘッズアップ(2人)固定、3〜4人卓廃止
  - 配り 6/5/5、FL 14枚化
  - 名称を「ターボ」→「デュエル」に変更

## 検討

### 1. 許諾は法的に成立しない
ゲームのルール（アイデア・手順・遊び方）は著作権の保護対象外。保護されるのは
具体的な文章表現・図表・名称のみ。ブログ作者にルールの「許諾権」は存在しない。
→ 許諾を求める行為自体が、存在しない権利を前提にしている（ADR-0001 16-18行目と同じ結論）。

### 2. 許諾メールは逆効果
- 相手に拒否権があるかのような構図を自分から作る
- 拒否されると、法的に問題ないのに気まずい立場に
- 金銭・条件要求の余地を与える
- 「許諾が必要と本人も認識していた」記録が残る

### 3. ルール変更で依存はさらに低下
ADR時点の「ほぼ忠実」状態から、もはや別ゲームと言える差分。許諾の必要性は当時よりさらに薄い。

## 推奨アクション

- **許諾メール**: 送らない。
- **お礼／事後報告メール**: 任意（コミュニティ礼儀として）。送るならフレーミングに注意：
  - ❌「使わせてください」 → ⭕「ブログに着想を得て独自にアレンジ。ありがとうございました」
- **送る前の確認2点**:
  1. ブログの文章・表を逐語コピーしていないか（ここだけ法的に実体あるリスク）
  2. CreditsScreen は本作を「オリジナルゲーム」と表記（CreditsScreen.tsx:24）。
     ADR-0001 未解決事項#5「著者クレジット文面」は未対応。1行クレジットを入れるかは別途判断。
