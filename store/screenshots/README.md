# スクリーンショットの撮り方

ストアのスクショは**最終UIが固まってから撮る**のが鉄則（途中で撮ると作り直しになる）。
このフォルダに、撮った画像（PNG）を入れていく。

## 必須サイズ

| ストア | サイズ | 枚数 |
|---|---|---|
| App Store | iPhone 6.7"（1290×2796） | 最低3枚、最大10枚 |
| App Store | iPhone 6.5"（1242×2688）※6.7があれば省略可な場合あり | 任意 |
| Google Play | 縦長 9:16 など、各辺 320〜3840px | 最低2枚 |

## 撮るべき画面（推奨5枚）

1. タイトル画面（ロゴ）
2. ゲーム画面（カード配置中・役表示が見える状態）
3. ファンタジーランドのカットイン or 紫のFL画面
4. 精算画面（ポイントの動き）
5. ホーム画面（3つの対戦モード）

## 撮り方

### A. 実機（推奨・高品質）
1. `cd packages/app && eas build -p ios --profile preview`（または android）
2. 実機にインストールして各画面を表示
3. iPhone: 電源+音量上ボタン / Android: 電源+音量下ボタンでスクショ
4. 6.7"端末が無ければ、シミュレータ（iPhone 15 Pro Max）でも可

### B. ブラウザ（下書き・手軽）
1. `cd packages/app && npm run web`
2. Chromeで http://localhost:8081 を開く
3. DevTools（F12）→ デバイスツールバー → iPhone 14 Pro Max など縦長端末を選択
4. DevToolsの「Capture screenshot」で撮影
5. 解像度が足りなければ、撮った画像を引き伸ばす（審査は最低解像度を満たせば通る）

撮ったファイルは `01-title.png`, `02-game.png` … のように連番で本フォルダに保存する。
